Canvas学习
使用Canvas绘制图形
1.首先在网页中创建一个画布在HTML加入canvas标签


设定画布的高度和宽度,在HTML中生成画布，使用JS操纵元素生成想要的效果。

2.获取画布
var canvas = document.querySelector(‘.mycanvas’)
var width = canvas.width = window.innerWidth
var height = canvas.height = window.innerHeight
将窗口宽度和高度赋值给画布的宽度和高度。整个窗口就是画布。
使用getContext()方法获得2d或者3d画布。
var ctx = convas.getContext(‘2d’) ctx变量中保存了2d画布
ctx.fillStyle = ‘rgb(0,0,0)’ fillStyle属性使画布填充一个颜色
ctx.fillRect(0,0,width,height) fillRect()绘制一个覆盖整个区域的矩形
0，0 代表的意思是距离左上角的距离，width，height代表的是矩形的长和宽

fillStyle=’’填充颜色 接受一个字符串 fillRect()绘制一个矩形，接受4个参数
fillStyle = ‘rgba(1,1,1,0.25)’,前3个参数是颜色，最后一个参数是透明度，透明度越高，底层就越清晰

3.画线条
获取画布
var canvas = document.querySelector(‘.mycanvas’)
var ctx = canvas.getContext(‘2d’)

ctx.strokeStyle = ‘rgb()’ 设定线条颜色
ctx.lineWidth = 5 设定线条的像素，默认为1
ctx.strokeRect = () 4个参数，设定线条距离左上角距离，和线条的宽高

4.绘制路径
beginPath():在钢笔当前所在位置开始绘制，在新画布中起始位置为（0，0）
moveTo()：将港币移动至另一个点，将钢笔跳至该点。 ctx.moveTo(50,59)
fill()：填充通过为当前所绘制的路径区域填充一个颜色来形成图案 ctx.fill()
stroke():通过为当前绘制路径的区域描边，来绘制一个边框形状 ctx.stroke（）
