发表于 2019-03-02

1.盒模型
（1）在CSS样式中每一个元素在处在一个盒模型中，width和height设置内容区，里面包含了你想呈现的内容，一段文字或者一个图片等等。
（2）这个盒子还有3个属性：padding border margin

padding：内边距，指的是元素内容区和边框以内的空间大小，单位为px，顺序为上右下左。例如：padding：10px 20px 30px 40px.用于设定元素与你的内容之间的距离，注意不能使用负值，也可以使用百分比赋值。

border：边框，在元素周围创建边框，是一个边缘。使用，border-width，border-height等等设置。

margin：外边距，是设置元素与周围元素的距离属性，可以使用百分比赋值，是相对于父元素width或height的距离百分比，和padding一样，参数是从上右下左赋值

对于IE的盒子模型，与一般都不同，普通的盒子宽度=width+padding+border+margin，IE盒子模型宽度=width+margin

特殊情况：外边距合并：当两个垂直外边距相遇则合并成一个外边距，合成后的取两者中的大值。行内框，浮动框，绝对定位不会发生外边距合并。
2.定位 绝对定位和相对定位
块级元素：每个元素都是从新的一行开始，可以容纳内联元素和其他块级元素，盒模型属性都可以控制。
行内元素：和其他元素在一行上，盒子模型距离不能改变，只能容纳文本，或者其他内联元素
可以设置display：block属性改变元素的属性
普通文档流将浏览器窗体自上而下分成一行行，在每行按照从左到右排放元素。
position属性：static，relative，absoulte，fixed

static：元素框正常生成，块级元素生成矩形框，成为文档流的一部分，行内元素生成行框嵌套在父容器中

relative：元素框偏移某个距离，元素仍保持为定位前的形状，原本的占位空间保留。元素模型不能重叠，会导致它覆盖其它元素

absolute：元素框从文档流中删除，并相对于其包含块定位，原来的所占空间会关闭，元素定位后生成一个块级框，不论在原来文档流中是何类型的框。元素模型可以重叠，设置z-idnex参数。设置移动参数会影响其他元素的位置

fixed：类似于absolute，不过其包含块是视窗本身也就是窗口。

float：浮动，使元素脱离文档流向左或者向右移动，可以在一行中设置两个div标签。通常一个div标签占一行，overflow：auth或者none可以清除浮动

1.介绍
jQuery是一个JavaScript库，可以极大的简化JavaSript代码的编写
$()是选择器，相当于document.getElementByName…可以选择标签，（）里面可以是标签，class，id。
2.基础语法
所有的jQuery选择器是已美元符号开头。并且在$(document).ready(function(){……})函数中。这是为了防止文档在加载就绪之前就运行jQuery代码，在DOM（文档模型）加载之前就运行函数，操作可能失败。
3.基本操作
$(‘*’)通配选择符，选取所有元素。
$(‘p.color’),选取class为color的

标签
$(‘#id’)选取id
$(‘.class’)选取class
常用的有：hide() show()元素的显示和隐藏 fadeln()与fadeout()元素的淡入和淡出，add()与remove()方法，对元素或者class进行增加删除
append()在选择标签结尾添加内容，prepend()在选择标签开头添加内容，after()在被选元素之后插入内容，before()

parents()遍历父类元素，children()遍历子类元素(基于Dom树进行遍历)，siblings()方法进行水平遍历，next(),nextAll()

4.常用操作
通过对文档模型的操作，对页面实现各种动画效果，大雪飘落等。

（2）使用选择器选择这个标签，并且创建相同的标签，并使它的css等于雪花的css名。
（3）获得屏幕高度和宽度。设定初始位置为0（left和top）使用Math.random()生成随机数使雪花的位置随机出现（改变css中的位置）
（4）设定一个定时器，每隔10ms生成一个雪花并每次left和top+0.2px。添加判断，left和top超过屏幕大小则重新赋值为0.
（5） 建立一个for循环不断生成雪花。

webpack是一款打包工具，用于模块化编译JavaScript。可以使自己项目更加方便使用。
1.安装
首先安装nide.js运行环境，并配置环境变量
在你的项目根目录下 运行命令 npm install webpack –save
npm install webpack-cli –save
2.在你的项目文件夹下创建文件夹app并在其中创建index.js。新建dist文件夹，并配置package.json文件夹，确保private：ture，并且删除main属性

3.运行命令 webpack app/index.js dist/bundle.js
app/index.js是一个入口，dist文件夹是一个出口，执行命令后dist文件中会生成bundle.js。
webpack可以去除繁杂的依赖关系，使用webpack能够更好的管理你的代码。
